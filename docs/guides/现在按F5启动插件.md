## 🎯 立即测试插件

### 第 1 步：启动插件

**在当前 VS Code 窗口按 `F5` 键**

或者：
1. 点击左侧的 **调试图标**（虫子🐛）
2. 在顶部下拉框选择 **"运行扩展"**
3. 点击绿色 **播放按钮** ▶️

### 第 2 步：等待新窗口

- 会自动打开一个新的 VS Code 窗口
- 窗口标题显示：`[扩展开发主机]`
- 等待几秒让扩展加载完成

### 第 3 步：查看侧边栏

在**新窗口**（扩展开发主机）中：
- 查看左侧边栏
- 应该看到 **OpenSkills** 区域
- 包含两个视图：
  - 📁 **OpenSkills** - Skills 列表
  - 📋 **Proposals** - 提议列表

### 第 4 步：测试命令

在**新窗口**中按 `Ctrl+Shift+P`，输入 "OpenSkills"：

应该看到这些命令：
- `OpenSkills: Open Web UI` ⭐ **试试这个！会打开浏览器**
- `OpenSkills: Initialize`
- `OpenSkills: Open Panel`
- `OpenSkills: Refresh`
- `OpenSkills: Approve Proposal`
- `OpenSkills: Reject Proposal`
- `OpenSkills: Trigger Wake`

## 💡 如果按 F5 没反应

### 方法 1: 使用调试菜单

1. 点击顶部菜单 **Run** → **Start Debugging**
2. 或者 **Run** → **Run Without Debugging** (`Ctrl+F5`)

### 方法 2: 使用调试面板

1. 点击左侧 **调试图标**（🐛）
2. 顶部选择 **"运行扩展"**
3. 点击 **绿色播放按钮** ▶️

### 方法 3: 查看输出

如果有错误：
1. 按 `Ctrl+Shift+Y` 打开调试控制台
2. 查看错误信息
